# Простой таск-трекер

React + Supabase, тёмная тема, mobile-first.

## Локальный запуск

1. Установить зависимости: `npm install`
2. Создать файл `.env` по образцу `.env.example`, подставить `VITE_SUPABASE_URL` и `VITE_SUPABASE_ANON_KEY`
3. Запустить: `npm run dev`

## Supabase

1. Создать проект на [supabase.com](https://supabase.com)
2. В SQL Editor выполнить миграцию из `supabase/migrations/001_schema.sql`
3. В Authentication → Providers включить Email; при необходимости отключить "Confirm email" для входа без подтверждения почты.
4. Вход по email и паролю.

## Деплой

- **Фронт (Vercel):** подключить репозиторий, в настройках проекта задать переменные `VITE_SUPABASE_URL` и `VITE_SUPABASE_ANON_KEY`, собрать командой `npm run build`, корень — `dist`
- **Бэк:** данные и авторизация в Supabase; отдельный бэк на Railway для этого проекта не нужен

## Функции

- Вход по email и паролю
- Настраиваемое число дней (1–7), кнопки сдвига даты на 1 день
- На ширине &gt; 500px дни в один ряд по горизонтали, на меньшей — по вертикали
- У каждого дня: список задач (можно свернуть) и список выполненных (можно свернуть)
- Отдельный блок «Без даты» с задачами без даты
- У каждой задачи: 8 предустановленных цветов текста, подзадачи
- При выполнении задача анимированно появляется в списке «Выполненные»
